# Layout: Creazione guidata richieste passaggio 2

La finestra Creazione guidata richieste: passaggio 2 consente di configurare il layout del rapporto, la formattazione dell’intestazione e il filtro delle dimensioni per l’output. È possibile utilizzare un layout pivot per configurare la richiesta, in modo analogo alla creazione di una tabella pivot di Excel.

In alternativa, è possibile selezionare manualmente le celle in cui si desidera visualizzare i dati del rapporto utilizzando [Layout personalizzato](/help/analyze/report-builder/layout/configure-the-custom-layout.md).
