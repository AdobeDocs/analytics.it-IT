# Elementi del foglio di lavoro protetto

Se proteggete il foglio di lavoro (bloccando le celle), non potete creare, modificare o aggiornare le richieste. Rimuovete la protezione dal foglio di lavoro prima di usare Request Manager.

Se state aggiornando la richiesta e tentate di modificare una cella durante il processo di richiesta aggiornata, un messaggio di errore Excel segnala che il foglio di calcolo Ã¨ protetto. Al termine della procedura di aggiornamento della richiesta, potete accedere al foglio di calcolo e rimuovere la protezione temporanea dei fogli di calcolo.
