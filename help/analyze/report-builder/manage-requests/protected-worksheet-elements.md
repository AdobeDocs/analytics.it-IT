# Elementi del foglio di lavoro protetto

Se proteggete il foglio di lavoro (bloccando le celle), non potete creare, modificare o aggiornare le richieste. Rimuovere tutta la protezione dal foglio di lavoro prima di utilizzare Request Manager.

Se state aggiornando la richiesta e tentate di modificare una cella durante il processo di richiesta di aggiornamento, viene visualizzato un messaggio di errore Excel in cui si informa che il foglio di calcolo Ã¨ attualmente protetto. Al termine del processo di richiesta di aggiornamento, potete accedere al foglio di calcolo e la protezione temporanea del foglio di calcolo viene rimossa.
