# Cos&#39;è il Report Builder?

Adobe Report Builder è un componente aggiuntivo per Microsoft Excel. Consente di generare richieste personalizzate a partire dai dati di Adobe Analytics, che è possibile inserire nei fogli di lavoro Excel. Le richieste possono fare riferimento in maniera dinamica a celle presenti nel tuo foglio di lavoro, inoltre hai la possibilità di aggiornare e personalizzare la presentazione dei dati da parte del Report Builder.
