# Gestione dashboard

## Migrate a legacy dashboard {#task_64E4A29DB99644FA82EB3372243F1417}

Le dashboard legacy esistenti continueranno a essere eseguite e potrai comunque modificarle, scaricarle e programmarle; tuttavia, non potete più creare nuove dashboard legacy. Ti consigliamo vivamente di aggiornare dashboard legacy esistenti al nuovo formato dashboard.

<!-- 

t_dashboard_migrate_legacy.xml

 -->

>[!NOTE]
>
>Moving forward, consider using [Analysis Workspace projects](https://marketing.adobe.com/resources/help/en_US/analytics/analysis-workspace/) and their ability to be downloaded and scheduled.

Quando copiate la dashboard legacy, il sistema apre la dashboard legacy per la modifica, in cui potete aggiungere contenuto precedente o nuovo contenuto. Quando copiate una dashboard legacy, l'originale viene mantenuto nell'elenco delle dashboard precedenti.

>[!NOTE]
>
>Aggiungendo contenuto precedente a una dashboard viene creata una dashboard basata sulla funzionalità più recente del dashboard. Tuttavia, il minirapporto legacy potrebbe contenere dati basati sulla piattaforma dati precedente.

**Per migrare una dashboard versione 14. x precedente**

1. Click **[!UICONTROL Analytics]** &gt; **[!UICONTROL Components]** &gt; **[!UICONTROL Manage Dashboards]**.
1. In the [!UICONTROL Manage] column, under [!UICONTROL Legacy Dashboards], click **[!UICONTROL Copy to New Dashboard]**.

   La dashboard copiata viene aperta nell'editor di layout dashboard.

   See [Editing Dashboard and Reportlet Data](../../analyze/reports-analytics/dashboard.md#task_B460CCD70D9F40FCAC6BBC1C044CC460).

## Share a dashboard {#task_2776BDE3FBC64E66B78DB6D6A53AAF49}

Procedura che descrive il modo in cui un amministratore può condividere (o inviare push) un dashboard a più utenti. When you push dashboards to users, the dashboards become available in user's [!UICONTROL Shared Dashboards] menu.

<!-- 

t_dashboard_share.xml

 -->

1. In the [!UICONTROL Dashboard Manager], locate the dashboard, then enable **[!UICONTROL Shared]**.
1. Click **[!UICONTROL Push To Users]**.  ![](assets/push.png)

1. On the [!UICONTROL Push Dashboard] page, select the target users or click **[!UICONTROL Check All]**.
1. Fai clic su **[!UICONTROL Save]**.

If shared users of your dashboard cannot see changes you made in the dashboard, check your Dashboard Manager to see if the users have chosen the **[!UICONTROL Copy Me]** option. In caso contrario, non possono vedere gli aggiornamenti o le modifiche apportate dall'utente. To see all the changes/updates, shared users need to select the **[!UICONTROL On Menu]** option in the Dashboard Manager.

## Schedule a dashboard for delivery {#task_639361A35E044B2DB9D50FCABEC776FF}

In the [!UICONTROL Dashboard Manager], you can see whether a dashboard is scheduled for delivery, and edit the schedule. Le opzioni di consegna del dashboard sono identiche alle opzioni di consegna dei report.


<!-- 

t_dashboard_delivery.xml

 -->

1. Aprite una dashboard.
1. Click **[!UICONTROL More]** &gt; **[!UICONTROL Send]**.

   See [Schedule and Distribution](../../analyze/reports-analytics/scheduling.md#concept_4EA333DFC7FD4E9CA086385A3DA10BE9) for more information.

## Archive a dashboard {#task_3B6AB949F0EC4063A08F80F254992117}

Procedura che descrive come archiviare qualsiasi dashboard inviato come file PDF. Il sistema archivia il file archiviato per due anni oppure finché non raggiungete un limite massimo di 4 GB di rapporti archiviati.

<!-- 

t_dashboard_archive.xml

 -->

1. Aprite una dashboard.
1. Click **[!UICONTROL More]** &gt; **[!UICONTROL Send]**.
1. In the [!UICONTROL Email Report] group, enable **[!UICONTROL Archive]**.
1. Specify delivery options, then click **[!UICONTROL Send]**.

   Potete visualizzare dashboard archiviati nel dashboard Manager. Alternatively, open a dashboard and click **[!UICONTROL More]** &gt; **[!UICONTROL View Archive]**.
