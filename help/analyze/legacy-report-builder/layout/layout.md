# Layout: Creazione guidata richieste passaggio 2

{{legacy-arb}}

La finestra [!UICONTROL Request Wizard: Step 2] consente di configurare il layout del report, la formattazione dell&#39;intestazione e il filtro delle dimensioni per l&#39;output. È possibile utilizzare un layout pivot per configurare la richiesta, in modo analogo alla creazione di una tabella pivot di Excel.

In alternativa, è possibile selezionare manualmente le celle in cui si desidera visualizzare i dati del report utilizzando il [Layout personalizzato](/help/analyze/legacy-report-builder/layout/configure-the-custom-layout.md).
