# Report Suite Manager

Una suite di rapporti definisce il reporting indipendente e completo su un sito Web scelto, su un insieme di siti Web o su un sottoinsieme di pagine Web. Solitamente, una suite di rapporti è un sito Web, ma può essere un segmento globale in cui hai combinato diversi numeri di siti per ottenere un totale. Quando accedi a una soluzione Adobe Analytics, seleziona una suite di rapporti da usare (tranne quando utilizzi roll-up che combinano suite di rapporti). Inoltre, se desiderate eseguire rapporti per una parte del sito, una suite di rapporti può essere più piccola di un sito Web. Soluzioni Analytics aggregate e report su questi negozi di dati. L'amministratore suite di rapporti consente di definire le regole che determinano il modo in cui i dati vengono elaborati in una suite di rapporti.
