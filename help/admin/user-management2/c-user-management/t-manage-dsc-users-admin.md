# Gestione degli utenti Ad Hoc Analysis

1. Fai clic su **[!UICONTROL Analytics]** > **[!UICONTROL Admin]** > **[!UICONTROL User Management]**.
1. Fai clic su **[!UICONTROL Manage Ad Hoc Analysis Users]**.
