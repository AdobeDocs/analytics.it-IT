# Configurare Document Cloud con Adobe Analytics

Puoi abilitare Document Cloud Analytics per rendere disponibili dimensioni e metriche specifiche per PDF nella tua suite di rapporti.

**[!UICONTROL Analytics]** > **[!UICONTROL Admin]** > **[!UICONTROL Report Suites]** > **`<select report suite>`** > **[!UICONTROL Edit Settings]** > **[!UICONTROL Document Cloud Management]**

Quando si modifica una suite di rapporti, sono disponibili le seguenti opzioni di Gestione Document Cloud:

* [!UICONTROL **Rapporti di Document Cloud**]

  I rapporti specifici per i servizi di streaming multimediale sono disponibili nella pagina Media Reporting (Reporting multimediale). Per ulteriori informazioni su Media Reporting, vedere [Configurare la generazione di rapporti di Document Cloud](/help/admin/tools/manage-rs/edit-settings/document-cloud-config.md).

* [!UICONTROL **Classificazioni Document Cloud**]

  Le classificazioni specifiche di Document Cloud sono disponibili nella pagina Classificazioni di Document Cloud. Per informazioni sulle classificazioni, vedere [Classificazioni di conversione](/help/admin/tools/manage-rs/edit-settings/conversion-var-admin/conversion-classifications.md).
