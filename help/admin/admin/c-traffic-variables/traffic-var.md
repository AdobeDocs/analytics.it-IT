# Panoramica della variabile traffico (prop)

Le variabili di traffico di Custom Insight (o proprietà) consentono di correlare dati personalizzati con eventi specifici relativi al traffico. Le variabili prop sono incorporate nel codice di implementazione in ogni pagina del sito web.

Utilizza la pagina [!UICONTROL Traffic Variables] per abilitare, disabilitare o rinominare i rapporti sulle variabili di traffico. Una volta disattivato, il rapporto per una variabile di traffico non viene visualizzato nell’interfaccia utente di .

Consulta anche [prop](../../../implement/vars/page-vars/prop.md) nella guida utente Implementa .
