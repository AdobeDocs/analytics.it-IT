# Panoramica dei modelli delle suite di rapporti

I modelli delle suite di rapporti configurano le impostazioni più comuni per diversi tipi di suite di rapporti.

Per risparmiare tempo durante la creazione di una nuova suite di rapporti, puoi selezionare un modello simile alla configurazione della suite di rapporti progettata. La selezione di un modello di suite di rapporti non limita in alcun modo l’implementazione. Dopo la creazione della suite di rapporti, qualsiasi configurazione effettuata da un modello di suite di rapporti può essere modificata.
