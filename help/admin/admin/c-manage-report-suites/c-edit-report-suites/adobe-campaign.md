# Visualizzare i dati di Adobe Campaign in Adobe Analytics

[!UICONTROL **Analytics**] > [!UICONTROL **Amministratore**] > [!UICONTROL **Suite di rapporti**] > seleziona una suite di rapporti > [!UICONTROL **Modifica impostazioni**] > [!UICONTROL **Adobe Campaign**]

Puoi visualizzare i dati di Adobe Campaign Standard in Adobe Analytics. Per ulteriori informazioni, vedere [Rapporti standard di Adobe Campaign](/help/integrate/adobe-campaign.md).
