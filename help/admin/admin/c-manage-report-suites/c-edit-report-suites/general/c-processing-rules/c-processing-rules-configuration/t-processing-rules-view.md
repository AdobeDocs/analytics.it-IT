# Visualizzare le regole di elaborazione attive

Le regole di elaborazione attive vengono visualizzate in amministrazione.

1. Fai clic su **[!UICONTROL Analytics]** > **[!UICONTROL Admin]** > **[!UICONTROL Report Suites]**.
1. Seleziona una suite di rapporti.
1. Fai clic su **[!UICONTROL Edit Settings]** > **[!UICONTROL General]** > **[!UICONTROL Processing Rules]**.

   Sono elencate tutte le regole di elaborazione attive per la suite di rapporti.

[Visualizzare la cronologia delle regole di elaborazione](/help/admin/admin/c-manage-report-suites/c-edit-report-suites/general/c-processing-rules/c-processing-rules-configuration/t-processing-rule-view-history.md)
