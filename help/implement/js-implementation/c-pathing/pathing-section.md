# Passa da sezione a sezione

Se desiderate sapere in che modo i visitatori passano da una sezione all'altra sul sito, dovete prima accertarvi di aver aggiunto i tag alle sezioni con la variabile canale.

With this data in place, you must have pathing enabled on the *`channel`* variable. If a user starts on the home page and moves to the Sports section, then to the News section, this activity shows up in the [!UICONTROL Site Section] path reports.
