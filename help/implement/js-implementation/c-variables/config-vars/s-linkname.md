# s.linkName

Nome che verrà visualizzato nel rapporto personalizzato, di download o di uscita del collegamento. Troncato a 100 caratteri. Può essere impostato su qualsiasi stringa. Questo è il `pev2` parametro nella richiesta di immagine.

Se impostato con [`linkType`](https://docs.adobe.com/content/help/en/analytics/implementation/javascript-implementation/variables-analytics-reporting/config-var/linktype.html), una richiesta di immagine verrà inviata come collegamento di download, personalizzato o uscita.
