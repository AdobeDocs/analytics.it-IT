# Direttive sulla memorizzazione nella cache

Il file libreria javascript deve essere memorizzato nella cache nel browser dell'utente dopo la prima volta che l'utente carica una pagina.

I clienti Adobe devono garantire che i server Web siano configurati per usufruire di questa funzionalit√†. Ad esempio, accertatevi che l'impostazione NO-CACHE sia impostata su false. Inoltre, verificate che la data di scadenza sia sufficiente. Accertatevi che le cache proxy siano configurate con la configurazione corretta. La documentazione del server Web del cliente fornisce ulteriori informazioni.
