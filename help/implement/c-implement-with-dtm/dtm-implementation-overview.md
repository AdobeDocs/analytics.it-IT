# Panoramica sull'implementazione Gestione dinamica dei tag

Utilizza Gestione tag dinamica per gestire i tag e raccogliere e distribuire dati nei diversi sistemi di marketing digitale. Gestione tag dinamica fornisce un singolo livello di dati che invia dati da più fonti. Gestione tag dinamica consente anche la distribuzione reattiva dei contenuti specifici dell'utente.

Questa sezione di aiuto fornisce informazioni specifiche sull'utilizzo di Gestione tag dinamica per implementare Adobe Analytics. For detailed information about Dynamic Tag Management, refer to the [Dynamic Tag Management Product Documentation](https://marketing.adobe.com/resources/help/en_US/dtm/). For information about accessing DTM and common tasks when starting to use DTM, see [Getting Started](https://marketing.adobe.com/resources/help/en_US/dtm/get_started.html) in the Dynamic Tag Management Product Documentation.

For a detailed summary of the Dynamic Tag Management implementation steps, refer to [Deploy Adobe Analytics Using Dynamic Tag Management](https://marketing.adobe.com/resources/help/en_US/analytics/getting-started/add-adobe-analytics-dtm-tool.html) in Getting Started with Adobe Analytics.
