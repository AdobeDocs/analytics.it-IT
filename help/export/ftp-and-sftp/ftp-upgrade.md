# Aggiornamento dei server Adobe FTP e SFTP

Gli aggiornamenti ai server FTP e SFTP di Adobe possono introdurre nuove configurazioni che influiscono sugli script di automazione (spesso utilizzati per scaricare o caricare regolarmente i dati), che sono spesso create per fare affidamento su determinate impostazioni del server.

Ad esempio, viene introdotto un nuovo stato di successo che influisce su uno script che utilizza lo stato di successo per attivare una determinata azione. Adobe notifica in modo proattivo agli utenti tali modifiche alla configurazione. Se Adobe notifica che un aggiornamento del server FTP è imminente, controlla gli script di automazione per assicurarti che funzionino ancora correttamente.
