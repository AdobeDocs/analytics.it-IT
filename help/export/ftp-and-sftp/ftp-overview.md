# Utilizzare FTP e SFTP con Adobe CX Enterprise

Scopri come utilizzare FTP e SFTP per trasferire file tra computer e server.

Metodi per utilizzare l’FTP con le soluzioni e i prodotti Adobe:

* Carica set di dati di classificazione di grandi dimensioni in Analytics.
* Caricare gli attributi del cliente su CX Enterprise per utilizzarli nelle soluzioni.
* Utilizzare Analytics per creare e gestire origini dati basate su FTP, che sfrutta il trasferimento di file FTP per importare dati non in linea o storici nell&#39;organizzazione CX

Questi articoli non descrivono necessariamente i passaggi dell’utilizzo dell’FTP con ciascun prodotto. Consiglia invece le best practice per l’utilizzo dell’FTP con Adobe.
