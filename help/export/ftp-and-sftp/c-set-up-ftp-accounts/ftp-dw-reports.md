# Report consegnati di Data Warehouse

Data Warehouse ti consente di richiedere rapporti che mostrano relazioni avanzate tra i dati da dati non elaborati in base alle tue domande univoche.

FTP delivery, in this case, is available as a part of the [!UICONTROL Advanced Delivery Options] setup. Per utilizzare l'FTP hai bisogno di un account FTP interno o un account FTP Adobe. Adobe non configura automaticamente il tuo account FTP attraverso l'interfaccia data warehouse.
