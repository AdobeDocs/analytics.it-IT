# Report consegnati del Report Builder

Utilizzare il componente aggiuntivo Adobe Report Builder per Microsoft Excel per creare richieste personalizzate a partire dai dati di analisi Web, che è possibile inserire nei fogli di lavoro di Excel.

Le richieste possono fare riferimento in modo dinamico alle celle all&#39;interno del foglio di lavoro ed è possibile aggiornare e personalizzare la presentazione dei dati. Per FTP o SFTP, devi assicurarti che l&#39;host sia disponibile esternamente. Per ulteriori informazioni sui report consegnati [!UICONTROL Report Builder], vedere [Guida di Report Builder](/help/analyze/report-builder/rb-overview.md).

Per informazioni sui limiti FTP o SFTP e sulla conservazione dei dati, consulta [Limiti FTP e SFTP e conservazione dei dati](/help/export/ftp-and-sftp/ftp-limits.md).
