# Conservazione dei dati e limitazioni di FTP e SFTP

Tutti gli account FTP e SFTP di Adobe hanno un limite di archiviazione dei dati di 100 GB. I file vengono conservati su questi account FTP per 90 giorni.

## Limiti sulle connessioni simultanee

* Non più di 50 connessioni simultanee al server per IP di origine
* Non più di 20 nuove connessioni in 60 secondi al server per IP sorgente
