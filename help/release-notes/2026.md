# Note sulla versione 2026 di Adobe Analytics

Scopri gli ultimi aggiornamenti sulle versioni dei [prodotti Adobe Experience Cloud](https://business.adobe.com/it/products/adobe-experience-cloud-products.html). Scopri le risorse di supporto autonomo più recenti con documentazione, tutorial e corsi su Experience League.

## Febbraio 2026 {#feb26}

| Funzione e descrizione | [Avvio del rollout](releases.md) | [Disponibilità generale](releases.md) |
| ----------- | ------- | ---- |
| **Feed dati migliorati** <br/>Sono ora disponibili i seguenti miglioramenti per i feed dati:<ul><li>Esperienza utente migliorata.</li><li>Nuova esperienza per la creazione e la gestione di modelli di colonna.</li><li>Ora puoi scegliere quando creare un modello di colonna da riutilizzare in feed di dati futuri. È inoltre possibile eliminare, modificare e copiare i modelli.<br/>In precedenza, ogni feed di dati creato generava un nuovo modello di colonna, che generava un numero elevato di modelli di colonna inutilizzati e non consentiva di eliminarli o gestirli.</li><li>Aggiungi e filtra per tag.</li><li>Visualizzare la cronologia dei processi di feed dati. (quando ha avuto inizio il periodo della richiesta, quando ha avuto inizio, quando ha finito e così via).</li><li>Inviare nuovamente o rielaborare i feed di dati. (Dalla pagina Job historyi)</li><li>Consenti hit in arrivo. Ora puoi includere i dati arrivati dopo che il processo di feed dati ha completato l’elaborazione dei dati all’interno della frequenza di reporting impostata.</li><li>Quando si sceglie una data di fine per un feed dati, la data di fine scelta viene inclusa come ultimo giorno del feed dati.<br/>In precedenza, la data di fine veniva esclusa dal feed di dati.</li><li>È ora possibile esportare la frequenza di 15 minuti, ma non è disponibile per impostazione predefinita. Affinché questa opzione sia disponibile nel tuo ambiente, devi prima contattare l&#39;Assistenza clienti Adobe e richiedere che la suite di rapporti sia configurata per supportare le esportazioni della durata di 15 minuti.</li></ul><p>**Nota:** con questi miglioramenti, vengono aggiornati anche gli URL delle pagine dei feed dati in Adobe Analytics. Aggiorna eventuali segnalibri esistenti per puntare alle nuove pagine di feed di dati entro il 30 aprile 2026.</p><p>Per ulteriori informazioni, vedere [Creare un feed dati](/help/export/analytics-data-feed/create-feed.md) e [Gestire feed dati](/help/export/analytics-data-feed/df-manage-feeds.md).</p> | mercoledì 20 gennaio 2026 | mercoledì 24 febbraio 2026<p>Rilascio originariamente pianificato per il mercoledì 10 febbraio 2026</p> |
| **Nuova guida API per suite di rapporti di Adobe Analytics 2.0** <p>Crea, recupera, aggiorna ed elimina le API della suite di rapporti. Per ulteriori informazioni, vedere la [guida di riferimento per l&#39;API della suite di rapporti di Adobe Analytics 2.0](https://developer.adobe.com/analytics-apis/docs/2.0/apis/report-suites/) e la [guida dell&#39;endpoint dell&#39;API della suite di rapporti di Adobe Analytics 2.0](https://developer.adobe.com/analytics-apis/docs/2.0/guides/endpoints/report-suites/).</p> | | Febbraio 2026 |
| **Nuova guida al rapporto sul rilevamento delle anomalie API di Adobe Analytics 2.0** <p>Creare un rapporto di rilevamento automatico delle anomalie API. Per ulteriori informazioni, vedere la [Guida dell&#39;endpoint Report API di rilevamento anomalie](https://developer.adobe.com/analytics-apis/docs/2.0/guides/endpoints/reports/anomaly/).</p> | | Febbraio 2026 |
| **Nuova guida ai rapporti con tendenze sulla data di base dell&#39;API di Adobe Analytics 2.0** <p>Creare un report KPI con tendenze data di base API automatizzata Per ulteriori informazioni, vedere [Guida dell&#39;endpoint report API per il rilevamento delle anomalie](https://developer.adobe.com/analytics-apis/docs/2.0/guides/endpoints/reports/kpi/).</p> | | Febbraio 2026 |
