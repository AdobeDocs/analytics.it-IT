# Tipo di browser

La dimensione &#39;Tipo browser&#39; elenca le organizzazioni che hanno creato il browser utilizzato dal visitatore. Questa dimensione è utile quando si desidera vedere quali browser globali utilizzano i visitatori. Fornisce valore sulla dimensione &quot;Browser&quot;, in quanto non elenca versioni diverse dello stesso browser come elementi dimensione separati.

## Compilare questa dimensione con i dati

Questa dimensione fa riferimento a una tabella di ricerca interna ad Adobe. Il valore di ricerca è basato sull’intestazione `User-Agent` HTTP nelle richieste di immagini. Se utilizzi una libreria AppMeasurement (ad esempio tramite  lancio del Adobe Experience Platform), questa dimensione non è disponibile.

## Elementi dimensione

Gli elementi dimensione includono le organizzazioni che creano browser. Gli elementi di dimensione comune includono `"Google"` (i creatori di [!DNL Chrome]), `"Apple"` (i creatori di [!DNL Safari]), `"Microsoft"` (i creatori di [!DNL Edge]), e `"Mozilla"` (i creatori di [!DNL Firefox]).
