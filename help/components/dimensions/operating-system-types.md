# Tipi di sistemi operativi

La dimensione &quot;Tipi di sistema operativo&quot; mostra il sistema operativo globale utilizzato dal visitatore, indipendentemente dalle versioni specifiche. Questa dimensione è utile per capire non solo quale sistema operativo e versione specifici è più comune, ma quale piattaforma OS tipica utilizza i visitatori.

## Popolare questa dimensione con i dati

Questa dimensione fa riferimento a una tabella di ricerca interna all’Adobe. Il valore di ricerca si basa sull’intestazione HTTP `User-Agent` nelle richieste di immagini. Se utilizzi una libreria AppMeasurement (ad esempio tramite tag in Adobe Experience Platform), questa dimensione funziona automaticamente.

## Elementi Dimension

Gli elementi di Dimension includono il tipo di sistemi operativi utilizzati. Gli esempi includono `"Microsoft Windows"`, `"Apple Macintosh"`, `"Google Android"` e `"Apple iOS"`.
