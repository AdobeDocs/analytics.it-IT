# Visitor ID

La &#39;ID visitatore&#39; [dimensione](overview.md) fornisce l&#39;identificatore univoco per ogni visitatore.

>[!IMPORTANT]
>
>Questa dimensione è disponibile solo in Data Warehouse.

## Popolare questa dimensione con i dati

Adobe genera automaticamente un ID visitatore per ogni visitatore. Questo valore è uguale al valore concatenato delle colonne `visid_high` e `visid_low` nei feed di dati. Per ulteriori informazioni, vedere [Riferimento colonna dati](../../export/analytics-data-feed/c-df-contents/datafeeds-reference.md).

## Elementi dimensionali

Gli elementi Dimension includono l’identificatore univoco di ogni visitatore.
