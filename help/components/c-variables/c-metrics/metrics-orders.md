# Ordini

Il numero di ordini effettuati sul tuo sito Web nel corso del periodo di tempo selezionato. Potete suddividere singoli periodi di tempo in base ad altre metriche per mostrare gli elementi (quali prodotti o campagne) che hanno contribuito al maggior numero di ordini durante tale intervallo.

| Usi | Descrizione |
|---|---|
| Traffic (Traffico) | N/D |
| Conversione   | *`Order`* è il numero di volte in cui l'evento di acquisto è impostato. |
| Ad Hoc Analysis | *`Order`* è il numero di volte in cui l'evento di acquisto è impostato. |
