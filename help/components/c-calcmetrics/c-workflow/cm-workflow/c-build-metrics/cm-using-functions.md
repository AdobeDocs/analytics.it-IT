# Usare le funzioni

Le funzioni ti consentono di filtrare/ordinare i dati ed eseguire analisi statistiche.

Per un elenco di tutte le funzioni, vedere [Funzioni di base](/help/components/c-calcmetrics/cm-reference/cm-functions.md) e [Funzioni avanzate](/help/components/c-calcmetrics/cm-reference/cm-adv-functions.md).

>[!NOTE]
>
>Quando includi funzioni in una formula di metriche calcolate, applica sempre la funzione prima di trascinare metriche o segmenti.
